## 몬스터 키우기

게임 구성 : 
    포켓몬 선택.  
    포켓몬 이름 짓기 완료 후 플레이어 포켓몬 등록됨  
    포켓몬 대련을 통해 포켓몬 성장시켜야함.  
    연구소에서 간식, 포켓몬 볼 등 판매  
    대련에서는 상대 포켓몬이 랜덤으로 설정됨. 물 > 불 > 풀 > 물   
    돌아가면서 공격 & 방어가 이루어짐. 공격차례에 공격 성공하면 상대 포인트 차감.  
    방어 성공하면 타격 x  
    체력이 방전되면 패배. 승리 시 포인트 지급  
    경험치가 다 차면 레벨업 성공 후 게임 종료.

## :checkered_flag: 기능 구현 목록

- 포켓몬 
  - [x] 이름, 타입, exp, 체력, 묘기가능횟수, 디테일 정보(색, 키, 몸무게) 필드를 가지고 있다.
  - [x] 묘기 부리기 메서드
  - [x] 간식 먹기 메서드 
  - [x] 울기 메서드
  - [x] 공격 당함 메서드
  - [x] 포켓몬 정보 제공 메서드
  - [x] 자식 클래스로 모부기, 불꽃숭이, 팽도리를 가진다.
        - [x] 모부기
            - [x] 디테일 정보(타입(풀), 초록, 0.4m, 10.2kg)를 가진다.
            - [x] 풀을 날리는 묘기를 부린다.
            - [x] 꾸르르르륵 하고 운다.
                - [x] 수풀부기
                    - [x] 풀타입, 디테일 정보(초록, 1.1m, 97kg)를 가진다.
                    - [x] 쿠르르르륵 하고 운다.

        - [x] 불꽃숭이
            - [x] 디테일 정보(타입(불)갈색, 0.5m, 6.2kg)를 가진다.
            - [x] 철주먹을 날리는 묘기를 보여준다.
            - [x] 우끼끼 하고 운다.
                - [x] 파이숭이
                    - [x] 불타입, 디테일 정보(갈색, 0.9m, 22kg)를 가진다.
                    - [x] 후키키 하고 운다.

        - [x] 팽도리
            - [x] 디테일 정보(타입(물), 파랑, 0.4m, 5.2kg)를 가진다.
            - [x] 물회오리를 만드는 묘기를 보여준다.
            - [x] 삐약삐약 하고 운다.
                -[x] 팽태자
                    - [x] 물타입, 디테일 정보(파랑, 0.8m, 23kg)를 가진다.
                    - [x] 페에엥 하고 운다.

    -[x] 디테일 정보
        - [x] 이름, 타입, 색, 키, 몸무게

- [x] 연구실
    - [x] 판매하는 상품의 리스트를 가지고 있다.
    - [x] 판매하는 상품의 정보를 반환하는 메서드
        - [x] 상품
            - [x] 상품 이름과 가격, 재고, 설명을 필드로 가지고 있다.

- [x] 플레이어
    - [x] 이름, 보유자산, 보유 상품 맵, 펫을 필드로 가진다.
    - [x] 펫 등록 메서드
    - [x] 상품을 구매하는 메서드
    - [x] 승리 포인트를 업데이트 하는 메서드
    - [x] 상품 사용 메서드
    
- [x] 체육관
    - [x] 내 포켓몬, 상대 포켓몬, 공격데미지, 피해데미지, 현재공격턴을 필드로 가진다.
    - [x] 랜덤 포켓몬 선정 메서드
    - [x] 공격 or 방어 위치를 정하는 메서드
    - [x] 공격&방어 결과 반환 메서드 (fight)
        - [x] 결과 (DTO)
            - [x] isAttack, isSuccess, 내가 선택한 위치, 상대 선택 위치, 내 포켓몬 남은 체력, 상대 남은 체력
    - [x] 게임 종료 확인 메서드
    - [x] 최종 결과 반환 메서드
        - [x] isWinner, point, exp

- [x] InputView
    - [x] 게임 시작 사용자 이름 입력받기
    - [x] 분양 받을 포켓몬 입력받기
        - [x] 분양 확인 or 취소 입력받기
    - [x] 포켓몬 이름 입력받기
    - [x] 메인 메뉴 선택 입력받기
    - [x] 상품 구매 입력받기
    - [x] 체육관 메뉴 입력받기
    - [x] 공격 or 방어 위치 입력받기

- [x] OutPutView
    - [x] 포켓몬 등록 완료 문구 출력하기
    - [x] 체육관 설명 출력하기
    - [x] 대련 시작 문구 출력하기
    - [x] 공격/방어 결과 출력하기
    - [x] 대련 결과 출력하기
    - [x] 포켓몬 진화 출력하기
    - [x] 게임 종료 문구 출력하기

    - [] 플레이어 이름은 1 이상 10 이하의 문자열이다.
    - [] 포켓몬 이름은 1 이상 10 이하의 문자열이다.
    - [] 메뉴 선택은 숫자로만 입력 가능
