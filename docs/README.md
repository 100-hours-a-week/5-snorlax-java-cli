## 몬스터 키우기

게임 구성 : 
    포켓몬 선택.
    포켓몬 이름 짓기 완료 후 플레이어 포켓몬 등록됨
    포켓몬 대련을 통해 포켓몬 성장시켜야함.
    연구소에서 간식, 체력회복제, 생명회복제 판매
    대련에서는 상대 포켓몬이 랜덤으로 설정됨. 물 > 불 > 풀 > 물 
    돌아가면서 공격 & 방어가 이루어짐. 공격차례에 공격 성공하면 상대 포인트 차감.
    방어 성공하면 타격 x
    체력이 방전되면 패배. 승리 시 포인트 지급
    포켓몬 간식을 줄 때 포켓몬의 행복한 모습을 볼 수 있음.
    경험치가 다 차면 레벨업 성공 후 게임 종료.

## :checkered_flag: 기능 구현 목록

- 포켓몬 
  - [] 이름, 타입, exp, 체력, 디테일 정보(색, 키, 몸무게) 필드를 가지고 있다.
  - [] 간식 먹기 메서드 
  - [] 묘기 부리기 메서드
  - [] 울기 메서드
  - [] 포켓몬 정보 제공 메서드
  - [] 자식 클래스로 모부기, 불꽃숭이, 팽도리를 가진다.
        - [] 모부기
            - [] 풀타입, 디테일 정보(초록, 0.4m, 10.2kg)를 가진다.
            - [] 간식을 먹으면 행복한 모부기 모습을 출력한다.
            - [] 풀을 날리는 묘기를 부린다.
            - [] 꾸르르르륵 하고 운다.
        - [] 불꽃숭이
            - [] 불타입, 디테일 정보(갈색, 0.5m, 6.2kg)를 가진다.
            - [] 간식을 먹으면 행복한 모부기 모습을 출력한다.
            - [] 철주먹을 날리는 묘기를 보여준다.
            - [] 우끼끼 하고 운다.
        - [] 팽도리
            - [] 물타입, 디테일 정보(파랑, 0.4m, 5.2kg)fmf rkwlsek.
            - [] 간식을 먹으면 행복한 팽도리 모습을 출력한다.
            - [] 물회오리를 만드는 묘기를 보여준다.
            - [] 삐약삐약 하고 운다.
    -[] 디테일 정보(DTO)
        - [] 색, 키, 몸무게

- [] 연구실
    - [] 판매하는 상품의 리스트를 가지고 있다.
    - [] 판매하는 상품의 정보를 반환하는 메서드
    - [] 상품(DTO)
        - [] 상품 이름과 가격을 필드로 가지고 있다.

- [] 플레이어
    - [] 이름, 보유자산, 보유 상품 맵, 펫을 필드로 가진다.
    - [] 펫 등록 메서드
    - [] 상품을 구매하는 메서드
    - [] 승리 포인트를 업데이트 하는 메서드
    - [] 상품 사용 메서드
    - [] 
    
- [] 체육관
    - [] 내 포켓몬, 상대 포켓몬, 현재공격턴을 필드로 가진다.
    - [] 대련 시작하기 메서드
    - [] 랜덤 포켓몬 선정 메서드
    - [] 공격 or 방어 위치를 정하는 메서드
    - [] 공격&방어 결과 반환 메서드
        - [] 결과 (DTO)
            - [] isAttack, result, 내 포켓몬 남은 체력, 상대 남은 체력, isEndGame

- [] InputView
    - [] 게임 시작 사용자 이름 입력받기
    - [] 분양 받을 포켓몬 입력받기
        - [] 분양 확인 or 취소 입력받기
    - [] 포켓몬 이름 입력받기
    - [] 메인 메뉴 선택 입력받기
    - [] 상품 구매 입력받기
    - [] 체육관 메뉴 입력받기
    - [] 공격 or 방어 위치 입력받기

- [] OutPutView
    - [] 포켓몬 정보 출력하기
    - [] 포켓몬 등록 완료 문구 출력하기
    - [] 체육관 설명 출력하기
    - [] 대련 시작 문구 출력하기
    - [] 대련 결과 출력하기
    - [] 포켓몬 진화 출력하기
    - [] 게임 종료 문구 출력하기
    
    
- [] validate
    - [] 플레이어 이름은 1 이상 10 이하의 문자열이다.
    - [] 포켓몬 이름은 1 이상 10 이하의 문자열이다.
    - [] 메뉴 선택은 숫자로만 입력 가능
